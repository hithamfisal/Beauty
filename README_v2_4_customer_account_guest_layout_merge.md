# Beauty Home Service v2.4 - Customer Account / Guest Booking Merge

تم دمج خاصية اختيار العميلة بين تسجيل الدخول برقم الهاتف أو المتابعة كضيف داخل الواجهة الحالية بدون استبدال تخطيط الواجهة.

## المنفذ

- إضافة اختيار طريقة الحجز في واجهة العميل:
  - تسجيل / دخول برقم الهاتف
  - المتابعة كضيف
- عند الدخول بالحساب يتم تعبئة الاسم ورقم الجوال والعنوان الافتراضي تلقائياً إن وجد.
- عند المتابعة كضيف يتم إدخال الاسم والجوال والعنوان داخل الطلب.
- إضافة تسجيل سريع بالهاتف و OTP داخل صفحة الحجز.
- تضمين `booking_customer_mode` في طلب الحجز بالقيمة `account` أو `guest`.
- إصلاح تسجيل دخول الأدمن في Backend ليتزامن مع متغيرات Vercel وقاعدة البيانات.
- إزالة مسار التشخيص المؤقت `/api/debug/admin-env`.

## بعد الاستبدال

1. شغّل Backend محلياً.
2. شغّل Customer Web محلياً.
3. اختبر الحجز كضيف.
4. اختبر الحجز بعد تسجيل الدخول برقم الهاتف.
5. ارفع إلى GitHub ثم أعد نشر Backend وبعده Customer Web.
